About

  • This project is the final project for MAIS-202: Accelerated Machine Learning Bootcamp by the McGill Artificial Intelligence Society.
  • Movie Plot Genre Classifier classifies the plot summaries using a binary classification for eight different genres: Action, Adventure, Comedy, Crime, Family, Mystery, Romance, and Thriller.
  • Movie Plot Genre Classifier was built using Keras, Scikit-learn, and Flask.
  • The datasets used to train the model were CMU Movie Summary Corpus Dataset while the IMDB Dataset was added because the preliminary results showed overfitting due to a lack of data.
  • The model used to make predict is a Bidirectional LSTM while the features used for the model extracted were from the 10000 most common words and Keras Tokenizer API for the sequencing.
